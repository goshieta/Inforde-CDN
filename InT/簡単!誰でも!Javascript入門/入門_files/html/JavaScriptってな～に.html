JavaScriptってな～に？

このページにたどり着いている以上たいていの人はJavaScriptが何かということをご存じだと思いますが、一応復習だと思って確認しておきましょう

JavaScriptはWEBページでよく使われる軽量なプログラミング言語です。WEBページ上で動的な動作をさせたいときに用いられます。そんなこと言われたところでわからないでしょうから、例をあげましょう。もしあなたが遊びで呪いのページを作りたいとします。ある程度スクロールされたら呪いの音が流れ、画面が勝手に上にスクロールされその途中の文字はすべて呪いの文字に変換されている。これを作る場合、まずある程度スクロールされた時を感知しなければなりません。これはJavaScriptを使います。そして次に呪いの音を流さなければなりません。これもJavaScriptが用いられます。そして文字の置き換えもJavaScriptが用いられます。このようにページを動かないページから動くページに変えるうえでJavaScriptはとても重要な役割を果たすのです。

またこのサイトでは書きませんがNode.jsなどを用いると、ブラウザ上だけではなく例えばサーバーなどでも使用することができます。

またJavascriptは比較的簡単なためプログラミング初心者に向いているといわれます。かくいう私も、初めて学んだプログラミング言語は,JavaScriptでした。とはいっても最初のほうは苦労しますがね...

たまにJavaScriptとJavaを混同している方がいらっしゃいますがJavaScriptとJavaは別の言語です。ただJavaScriptはJavaをもとにして開発されたという歴史があります。私はJavaをかけないので(守備範囲はC++,C,Python)よく知りませんが、文法も目的の分野も違います。たとえるならワインとぶどうジュースといったところでしょうか(←下手)。ネット上を探していたら「メロンとメロンパン」といういいたとえがありました。まあそんな感じです。

環境構築

ほかの言語の経験者ならご存じでしょうが、初心者プログラマーはまず環境構築でつまずきます。場合によってはそれが原因であきらめることも...しかし、安心してください！JavaScriptは環境構築がほとんど必要ないです。ここはHTMLとCSSと一緒ですね。しいて言うなら、ブラウザによって動きが異なることがあるのでたくさんブラウザをダウンロードしてください。Google Chrome,Microsoft Edge,Opera,IE,Firefox,Safariあたりがあればいいです。そうはいってもIEはもうサポートが終了しましたし、最近はどのブラウザも動きが統一されてきたので最低限Google Chromeさえあれば事足りるかと思います。（ただSafariがちょっと厄介）

エディタ（プログラムを書くときに使うもの）はメモ帳でも書くことはできますがVisualStudioCodeがあったほうがいいかなと思います。ダウンロードはこちらから。こちらのサイトにダウンロード手順がまとめてあります。

学ぶ上での心得

これに関してはほかのどの言語にも共通することですが、プログラミング言語は学ぶだけではなく実践が大事です。パソコン１台さえあれば簡単に実践できるのでぜひ積極的に実践してください。最初のほうはこのページに書いてあるコードをコピペして動くのを確認するだけでもいいです。そして慣れてきたらコピペするだけではなく自分なりに手を加えてみたり、例以外のものを作ってみたりしましょう。それが習得への近道です。またこのページにあるコードは自由に使ってもらって構わないです。（どうせ大したこと書いていないんで）

最後にこの講座がどのように進むかということだけ確認しておきます。この講座の最終目標は一通りJavaScriptが書けるようになるです。基本的に基本編と発展編さえやればこの目標は達成できるように作ってあります。実践編はとりあえず勉強してみたけどいまいち何を作れるのかわからない人向け。GAS編は追加でGASも学びたい人向け、HTML編とCSS編はこのサイトで学習するうえで最低限必要なHTMLとCSSの知識を紹介しています。まあそういうことなので頑張ってください。それではまた次の記事で。
