条件分岐

今回のタイトルは条件分岐です。名前の通り条件によって結果が変わります。それが条件分岐です。頑張っていきましょう

比較演算子

まずは比較演算子について学びます。「なんだよそれ」という方が大半かもしれませんが比較演算子は左側の値と右側の値を比べてTrueまたはFalseを返します。

とりあえず例をみましょう。主に使われる比較演算子を表にまとめました。

==値が等しい場合Trueを等しくない場合はFalseを返します。代入演算子の=と間違えやすいので気を付けましょう。
<左の値より右の値のほうが大きい場合Trueを返す。等しかったり小さかった場合はFalse。
>さっきの逆。
<=右の値が左の値よりも大きいまたは等しいときにTrueを返す。小さい場合はFalse
>=さっきの逆

ざっとこんな具合です。では練習問題です。下のコードはそれぞれログになんて出力されるでしょうか？


                console.log(12==7)
                console.log(34>=67)
                console.log(22*54+32==1220)

答えは上から順にfalse,false,trueです。あっていましたか？

if文

いよいよ本題の条件分岐に入っていきます。Javascriptでは条件分岐させる際if文を使います。使い方を見てみましょう。


                if(条件式){
                    //コードをここに記述
                }

条件式のところには、先ほど学んだ比較演算子を使った式を入れます。Trueが返ってきた場合、if文の中に書いてあるコードが実行されfalseが帰ってきた場合、if文の中のコードは無視され次に進みます。例を見てみましょう。


                let g=12
                if(g>10){
                    console.log('gは10よりも大きいです。')
                }
                console.log("プログラムを終了します。")

これを実行すると


                >>gは10よりも大きいです。
                >>プログラムを終了します。

としっかり出力されます。値をいろいろと変えて試してみてください。しかしこの場合gが10より大きいということしか教えてくれません。10以下の時もログに何か出力させたいですよね。そんな時に使うのがelseです。これを使うことでfalseが帰ってきたときの処理を記述することができます。さらにelse ifでelseの中で分岐させることもできます。早速書いてみましょう。


                let aha=12
                if(aha<10){
                    console.log('ahaは10よりも小さい')
                }
                else if(aha==10){
                    console.log('ahaは10です。')
                }
                else{
                    console.log('ahaは10よりも大きい')
                }

実行すると無事「ahaは10よりも大きい」と出てきました。ほかにもいろいろ値を変えて試してみてください。

これで条件分岐の回は終わりです。よく使うので覚えておいてください。
