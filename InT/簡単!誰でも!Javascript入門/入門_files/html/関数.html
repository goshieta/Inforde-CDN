関数

今回のテーマはズバリ！関数です。皆さんは関数といわれてわかりますか？数学で習ったことを覚えていらっしゃる方なら、「関数ってたしかxを一つに定めるとyも一つに定まるあれじゃなかったけ...」のような感じで思い出すのではないかと思います。しかし、プログラミングにおける関数はそれとは少し性質が違うので、しっかりと押さえておきましょう。

関数の宣言

早速関数について書いていくのですが、関数ってなんだという話ですよね。関数に関してはいろいろなとらえ方がありますが、とりあえず、コードの集合体という認識でいてください。関数の宣言方法ですが、次のように行います。


                function 関数名(引数){
                    //ここにコードを記述
                }

関数名は好きな名前にしていいです。できるだけ何をする関数かわかりやすい名前にしましょう。ただし変数の時と同じように一定のルールに従う必要があります。変数の時とほとんど同じなので復習としてみてください。

1. $と_以外の特殊記号は使うことができない。

2. 予約語を用いることはできない。(let,var,consoleなど)

3. 関数名の先頭に数字をつけてはならない。

そして引数ですがこれは、関数を実行するときに関数に渡す値のことです。これに関しては後ほど詳しく説明します。

ですが関数はとりあえず定義するだけでは実行することはできません。実行するには以下のように記述する必要があります。


                関数名(引数)

とりあえず今のままではわかりにくいと思うので、例を下に示します。


                function hello(){
                    alert('Hello world!')
                }

                hello()

これを実行すると画面にHello world!とメッセージが表示されます。

詳しく解説していきます。まず、最初のfunctionは関数を定義するときに書くものですね。letみたいなものです。そして次にhello()と書いてありますが、ここは例に、関数名(引数)と書いていた部分に当たります。helloは関数名で、そのあとに()を付けいていますが、本来はこの中に引数を記述しますが、今回は必要ないので省いています。そして関数の中のコードはalert('Hello world')ですね。これは解説する必要はないと思います。そして関数が終わった後にhello()と書いてありますね。これは、関数helloを実行するという意味です。これを書くことによりhelloが実行されるのですね。引数は必要ないので入れていません。

引数

ところでこの形どこかで見たことありませんか？これまでメッセージを表示するときにalert()を使用していましたが、実はあれも関数です。そしてこの関数に、表示させたいメッセージ例えば「江田島」などを「alert('江田島')」のような形で渡していましたね。これが引数です。関数に実行時に何らからの値を渡すことができるのです。例を示してみましょう。下の関数squは渡された値を3乗して返します。


                function squ(val){
                    return val**3
                }
                console.log(squ(5))
                console.log(squ(100))
                console.log(squ(4))

これを実行すると


                >>125
                >>1000000
                >>64

となります。まず最初に関数を定義します。関数名はsquです。そして()の中を見るとvalと書いてありますね。これは引数名です。自分で自由に設定することができます。そして関数の中のコードはreturn val**3となってます。returnは関数の呼び出しもとに値を返すことができます。この場合はval**3という値を返しています。つまり引数valの3乗ですね。関数の下にはconsole.log(squ(5))とあります。つまり、関数squを実行して引数valには5を与えているのですね。そしてval**3が帰ってくるので（この場合は5**3=125)それをconsole.log()でログに表示させているのです。

大体のイメージはつかんでいただいたでしょうか。これで今回は終わりです。ではまた皆さんどこかで会いましょう！
